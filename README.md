# MIDI-HA Bridge

**Trigger your Home Assistant lights, scenes, and scripts straight from any MIDI cue.**

No Node-RED, no MQTT broker, no YAML — just map a MIDI note to a Home Assistant action.

A lightweight macOS menu bar app that turns MIDI cues into Home Assistant actions.

Built for Proclaim® workflows - also tested with ProPresenter, EasyWorship, Logic Pro, and Ableton Live as a MIDI source.
