# MIDI-HA Bridge

**Trigger your Home Assistant lights, scenes, and scripts straight from any MIDI cue.**

No Node-RED, no MQTT broker, no YAML — just map a MIDI note to a Home Assistant action.

A lightweight menu bar / tray app for **macOS and Windows** that turns MIDI cues into
Home Assistant actions.

Built for Proclaim® workflows — also tested with ProPresenter, EasyWorship, Logic Pro,
and Ableton Live as a MIDI source.

This repo is the **download center**. Grab the latest build from the releases page:

### ⬇️ [Download →](https://github.com/freshcoffee/midi-ha-releases/releases)

- **macOS** — download the `.dmg` (Apple-notarized).
- **Windows 11** — download the `Setup .exe`. Windows has no built-in MIDI loopback, so
  install Microsoft's [Windows MIDI Services](https://microsoft.github.io/MIDI/) first — the
  app links you there and explains it in Settings if no MIDI ports are found.

For the setup guide, features, and support, visit the website:
